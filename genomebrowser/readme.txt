Genome Browser example in Protovis
==================================


by J. Christopher Bare
2010-03-10

A quick demo to try out the Protovis javascript visualization framework. Test how well Protovis holds up to large numbers of features by making the width larger.

Data in halo.chr.json comes from the NCBI Genomes database. Data in transcript.reference.json comes from tiling array experiments reported in "Prevalence of transcription promoters within archaeal operons and coding sequences", Koide et al., 2009.
